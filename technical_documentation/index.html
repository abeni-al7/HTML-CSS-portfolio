Introduction

Python is a high-level, interpreted programming language known for its readability and versatility. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Getting Started

Installation

Download and install Python from python.org.

Running Python Code

  • Interactive Mode: Open a terminal and type python or python3.
  • Script Mode: Save code in a .py file and run it with python filename.py.

Basic Syntax

Variables

x = 10
name = "Alice"

Data Types

  • Numbers: int, float, complex
  • Strings: "Hello"
  • Lists: [1, 2, 3]
  • Tuples: (1, 2, 3)
  • Dictionaries: {"key": "value"}
  • Sets: {1, 2, 3}

Comments

# This is a single-line comment
Control Structures

Conditional Statements

if x > 0:
    print("Positive")
elif x == 0:
    print("Zero")
else:
    print("Negative")

Loops

For Loop

for i in range(5):
    print(i)

While Loop

while x > 0:
    print(x)
    x -= 1

Functions

def greet(name):
    return "Hello, " + name print(greet("Alice"))
Object-Oriented Programming

Classes and Objects

A class is a blueprint of an object and an object is an instance of a class that represents a real world entity.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def greet(self):
        return "Hello, " + self.name
p1 = Person("Alice", 30) print(p1.greet())

File Handling python

Files can be opened and read using the following syntax:

# Writing to a file
with open('test.txt', 'w') as file:
    file.write("Hello, World!")

# Reading from a file
with open('test.txt', 'r') as file:
    content = file.read()
    print(content)
Error Handling

Python handles errors with the popular try/except block as shown here:

try:
    print(10 / 0)
except ZeroDivisionError:
    print("Cannot divide by zero")
Libraries and Frameworks

Some popular libraries include NumPy for Numerical computations, Pandas for Data analysis, Matplotlib for Plotting graphs, Requests for HTTP requests and Web Frameworks like Django and Flask.
